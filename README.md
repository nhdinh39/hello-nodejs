# Getting Started React-Simple-App

## Clone Repositories
```
    git clone https://github.com/huydinh39/react-simple-app.git
```

## Move to react-simple-app folder
```
    cd react-simple-app
```


## Install dependences
```
    npm install
```

## Run server
```
    npm start
```

## Check status

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
